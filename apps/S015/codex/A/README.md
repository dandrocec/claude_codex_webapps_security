# Countdown Label App

A small Node.js/Express app that lets a user enter an event label and target date, then displays a live countdown page for that event.

## Run locally

Install dependencies:

```bash
npm install
```

Start the app on port 5015:

```bash
npm start
```

Then open:

```text
http://localhost:5015
```

The app also respects a `PORT` environment variable if you need to override the default.
