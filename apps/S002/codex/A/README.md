# BMI Express App

A simple Node.js and Express web app that calculates body-mass index from height in centimeters and weight in kilograms.

## Run locally

Install dependencies:

```bash
npm install
```

Start the server on port 5002:

```bash
npm start
```

Open the app in your browser:

```text
http://localhost:5002
```

The app also respects a custom `PORT` environment variable, but defaults to `5002`.
