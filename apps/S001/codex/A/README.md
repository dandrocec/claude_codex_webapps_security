# Flask Greeting App

A tiny Flask app that asks for a person's name and displays a greeting.

## Run locally

1. Create and activate a virtual environment:

   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```

2. Install dependencies:

   ```powershell
   pip install -r requirements.txt
   ```

3. Start the app on port 5001:

   ```powershell
   python app.py
   ```

4. Open http://127.0.0.1:5001 in your browser.
