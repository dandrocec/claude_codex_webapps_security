# Photo Share Express

A small Node.js/Express photo-sharing app. Users can register, upload photos, follow other users, like photos, comment on photos, and browse a feed of recent photos from people they follow.

## Run Locally

```bash
npm install
npm start
```

Open `http://localhost:5073`.

The app stores SQLite data in `data/app.db` and uploaded images in `uploads/`. Both folders are created automatically on startup.
