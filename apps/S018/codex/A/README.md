# Password Strength Checker

A small Node.js/Express app where a user enters a candidate password and receives a weak, medium, or strong rating based on length and character variety. The response includes brief feedback for improving the password.

## Run locally

Install dependencies:

```bash
npm install
```

Start the app on port 5018:

```bash
npm start
```

Open the app in a browser:

```text
http://localhost:5018
```

You can also override the port if needed:

```bash
PORT=5018 npm start
```
