# Harbor Table Food Ordering

A Node.js/Express food-ordering app with a customer menu, cart checkout, staff order queue, server-computed totals, and SQLite persistence.

## Run locally

```bash
npm install
npm start
```

Open `http://localhost:5067` for the customer menu and `http://localhost:5067/staff` for the staff order screen.

The app stores data in `food_ordering.sqlite`, which is created automatically on startup.
