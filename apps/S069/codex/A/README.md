# Crowdfunding Express Site

A small Node.js/Express crowdfunding application. Creators can launch campaigns with funding goals and deadlines, and backers can pledge money toward active campaigns. Campaign data and pledges are stored in a local SQLite database.

## Run Locally

```bash
npm install
npm start
```

Open `http://localhost:5069` in your browser.

The app creates `data/crowdfunding.sqlite` automatically on first start.
