# Text Counter

A small PHP web app that accepts a block of text and reports the number of characters, words, and lines. The submitted text is echoed back below the results for reference.

## Run locally

Requires PHP 8.1 or newer.

```bash
php -S 0.0.0.0:5006 index.php
```

Then open:

```text
http://localhost:5006
```

If Composer is installed, you can also run:

```bash
composer run serve
```
