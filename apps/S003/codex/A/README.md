# Tip Splitter

A single-page PHP app that calculates the tip amount, total bill, and per-person amount from a bill amount, tip percentage, and number of people.

## Requirements

- PHP 8.1 or newer
- Composer is optional; there are no third-party dependencies.

## Run locally on port 5003

From this directory:

```bash
php -S 127.0.0.1:5003 index.php
```

Or with Composer:

```bash
composer run serve
```

Open http://127.0.0.1:5003 in your browser.
