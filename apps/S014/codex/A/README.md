# Inspirational Quote Flask App

A small Flask app that shows a random inspirational quote from a built-in list each time the page loads. Use the button on the page to fetch another random quote without refreshing.

## Run locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Start the app on port 5014:

```bash
python app.py
```

Open `http://127.0.0.1:5014` in your browser.
