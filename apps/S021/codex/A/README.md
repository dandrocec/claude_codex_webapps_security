# Base64 PHP App

A small PHP web app with a form for Base64 encoding and decoding text.

## Run locally

From the project directory, start the PHP built-in server on port 5021:

```bash
php -S 127.0.0.1:5021 -t public
```

Then open:

```text
http://127.0.0.1:5021
```

You can also use Composer's script:

```bash
composer run serve
```
